Calculate Mean Square Error

In this exercise, you will implement the formula for the Mean Square Error (MSE). Remember that the error is the difference between the predicted value and the actual target. Squaring ensures positive results. 

def calculate_mse(predictions, targets):
    # TODO: Calculate the squared difference
    n = len(predictions)
    sum_errors = sum((p - t)**2 for p, t in zip(predictions, targets))
    return sum_errors / n

print(f"MSE: {calculate_mse([1, 2, 3], [1, 2.5, 3.2])}")

Helper Notes

  • • MSE is always non-negative.
  • • A lower MSE indicates a better fit.
  • • The units of MSE are squared units of the target.
